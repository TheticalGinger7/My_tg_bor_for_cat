# My_tg_bor_for_cat
Этот бот создан для того чтобы определять породу котов пока только нескольких Сфинкс, Мейн-кун, Сиамская, Британская короткошерстная.
Чтобы его использовать через бота Father в телеграмме создаёте бота копируете токен и вставляете вместо надписи TOKEN дальше просто запускаете и всё.
